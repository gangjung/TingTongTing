# TingTongTing
<hr />

## 개요
>+ 움직이는 공을 마우스를 이용하여 만든 벽을 이용하여 공을 목적지로 이동시키는 게임입니다.

## 배경
>"윈도우즈 프로그래밍" 과목 프로젝트로, Win32 API를 이용하여 간단한 충돌처리를 구현해보기 위해 제작되었습니다.
